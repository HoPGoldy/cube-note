## 注册

- 在首次启动时要求输入用户名和密码。
- 后端只保存密码的 sha512 摘要，并使用 128 长度的字符串加盐。
- 该用户为管理员账号，可以在设置里新增用户并获取注册链接
- 新用户可以通过注册链接初始化账号

## 登录

- 每天错误登录次数超过3次，将会被锁定一天。

## 数据结构

- 笔记表
    - 名称
    - 创建日期
    - 更新日期
    - 内容
    - 下属笔记 id
    - 笔记关联 id

- 笔记关联表
    - 创建日期
    - 更新日期
    - 笔记A id
    - 笔记B id
    - 介绍

- 用户表
    - 用户名
    - 密码 sha512 摘要
    - 盐

## 功能

- 创建笔记，编辑笔记，删除笔记
- 关联笔记
- 注册、登录、邀请
